Overview of CST 438 - Software Engineering.

  • October 12, 2021
  • By Justin Nguyen

Course Description ..

Software Engineering is a comprehensive course that delves into the principles, methodologies, and practices involved in the development of high-quality software systems. Students will learn the fundamental concepts of software engineering, including requirements analysis, software design, coding, testing, and maintenance, to produce reliable and scalable software solutions.

Software Engineering: where we turn coffee into code and transform ideas into user-friendly magic.

In this course, students will explore software development processes, such as agile methodologies and the waterfall model, and learn how to apply them to real-world projects. They will gain hands-on experience with software engineering tools, version control systems, and collaborative development environments. Students will also learn about software quality assurance techniques, including testing methodologies and strategies for ensuring software reliability and performance.
